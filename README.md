# web3-training
learn how to write smart contract with solidity and javascript

### environment
* nodejs 16.11.0
* yarn 3
* hardhat
* Rinkeby network
* [Alchemy](https://www.alchemy.com/)
* [ganache](https://trufflesuite.com/ganache/)

```shell
npx hardhat help
npx hardhat test
GAS_REPORT=true npx hardhat test
npx hardhat node
npx hardhat run scripts/deploy.ts
```
